## 2. Дашборд

Главная страница с обзором всех проектов:

- **KPI-карточки:** количество активных проектов, визитов, открытых замечаний, загруженных фото
- **Лента активности:** последние действия по всем проектам (замечания, визиты, счета, комплектация)
- **Активные проекты:** карточки проектов с прогрессом и статистикой

Нажмите на карточку проекта, чтобы перейти к нему.

---

## 3. Проекты

### 3.1 Список проектов
Навигация: **Боковая панель > Проекты**

Все проекты отображаются карточками. Каждая карточка показывает:
- Название и адрес
- Количество визитов / фото / открытых замечаний
- Последняя активность
- Прогресс (%)

### 3.2 Создание проекта
Нажмите **"+ Новый проект"** в правом верхнем углу.

Заполните:
- **Название** (обязательно) — например, "Квартира на Тверской"
- **Адрес** — адрес объекта
- **Тип сценария** — "Блок" или "ГКЛ" (определяет этапы стройки)
- **Дата старта** — планируемая дата начала

### 3.3 Редактирование названия
На странице проекта нажмите иконку карандаша рядом с названием. Появится поле ввода:
- Введите новое название
- Нажмите **"Сохранить"** или Enter
- Нажмите **"Отмена"** или Escape для отмены

## 4. Страница проекта

Проект содержит 6 вкладок (видимость зависит от роли):

### 4.1 Обзор
- KPI проекта: визиты, фото, замечания, счета
- Информация: адрес, заказчик, дата старта, прогресс
- Последние визиты с быстрым переходом

### 4.2 Journal (Счета и визиты)
**Счета:**
- Список всех счетов с суммой, статусом и ссылкой на оплату
- Кнопка **"Выставить счёт"** открывает модалку:
  - Название (обязательно)
  - Сумма в рублях (обязательно, > 0)
  - Срок оплаты (необязательно)
  - Ссылка на оплату (необязательно) — URL из банковского приложения

Если ссылка на оплату указана, рядом со счётом появляется кнопка "Оплатить", открывающая ссылку в новой вкладке.

**Визиты:**
- Хронология завершённых визитов. Нажмите на визит, чтобы открыть его.

### 4.3 Визиты
- KPI: визиты по договору, завершённые, оставшиеся, запланированные
- Прогресс-бар
- Два раздела: "Запланированные" и "Завершённые"
- Кнопка **"Запланировать визит"**:
  - Название (обязательно)
  - Дата (обязательно)
  - Заметка (необязательно)

### 4.4 Supply (Комплектация)
Модуль управления позициями комплектации. Включает:
- **Dashboard** — сводка по позициям, статусам, рискам
- **Спецификация** — таблица всех позиций с категорией, сроком, бюджетом, статусом
- **Timeline** — Gantt-диаграмма сроков
- **Этапы** — этапы стройки (блок/ГКЛ)
- **Импорт** — загрузка позиций из Excel-файла (4-шаговый визард)
- **Настройки** — дисконт, параметры

### 4.5 Документы
- Карточки документов с форматом (PDF, DWG, XLSX, PNG)
- Нажмите на документ — откроется в новой вкладке
- Кнопка **"Загрузить"** — модалка с drag-and-drop:
  - Перетащите файл или нажмите для выбора (макс. 50 МБ)
  - Название (обязательно)
  - Версия (по умолчанию "1.0")
  - Формат определяется автоматически

### 4.6 Настройки
Две секции:
- **Роли и доступ** — список участников проекта, приглашение новых
- **Детали проекта** — даты, условия оплаты, параметры комплектации

**Приглашение участников:**
- Два способа: по email (существующий пользователь) или по ссылке (генерация уникальной ссылки)
- При приглашении выбирается роль и уровень доступа

## 5. Страница визита

Нажмите на визит в списке, чтобы перейти к деталям:

- **Заголовок** с датой, автором, статистикой (фото, замечания)
- **Фильтры:** Все / Принятые / Замечания
- **Фотокарточки** с зоной, комментарием, статусом
- **Смена статуса:** выберите новый статус в выпадающем списке прямо на карточке

### Загрузка фото
Нажмите **"Добавить фото"**:
1. Перетащите файл или нажмите для выбора (JPG, PNG, WebP, макс. 20 МБ)
2. Предпросмотр загруженного фото
3. Выберите **зону** (кухня, ванная, гостиная и т.д.)
4. Добавьте **комментарий**
5. Выберите **статус** (новое, замечание, принято, в работе)
6. Нажмите **"Загрузить"**

## 8. Импорт из Excel

### Подготовка файла
Excel-файл (.xlsx или .xls) должен содержать колонки с данными позиций комплектации.

### Шаги импорта

**Шаг 1: Загрузка файла**
Перетащите файл в зону или нажмите для выбора.

**Шаг 2: Маппинг колонок**
Система попытается автоматически определить соответствие колонок. Проверьте и скорректируйте:
- Название (обязательно)
- Категория
- Срок поставки (дни)
- Количество
- Поставщик
- Бюджет
- Заметки
- Этап

**Шаг 3: Предпросмотр**
Просмотрите первые 5 строк из загруженных данных.

**Шаг 4: Импорт**
Нажмите "Импортировать" и дождитесь результата.

## 9. Роли и уровни доступа

### Роли

| Роль | Описание |
|------|----------|
| designer | Дизайнер — полный доступ ко всему |
| assistant | Ассистент — полный доступ (кроме удаления проекта) |
| client | Заказчик — просмотр обзора, визитов, документов |
| contractor | Подрядчик — просмотр + загрузка фото |
| supplier | Комплектатор — только модуль Supply |

### Уровни доступа

| Уровень | Описание |
|---------|----------|
| view | Только просмотр |
| view_comment | Просмотр + комментарии |
| view_comment_photo | Просмотр + комментарии + загрузка фото |
| view_supply | Просмотр + операции с комплектацией |
| full | Полный доступ |

## 10. Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| Enter | Сохранить (в полях ввода, при редактировании названия) |
| Escape | Отменить (закрыть модалку, отменить редактирование) |
