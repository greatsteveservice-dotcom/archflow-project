# ЖАН — Журнал авторского надзора

Цифровая фиксация авторского надзора для дизайнеров интерьера и строительных компаний.

## Структура файлов

Замените/добавьте эти файлы в ваш репозиторий:

```
JAN/
├── tailwind.config.ts          ← ЗАМЕНИТЬ (добавлена тема zhan)
├── src/app/
│   ├── globals.css             ← ЗАМЕНИТЬ
│   ├── layout.tsx              ← ЗАМЕНИТЬ
│   ├── page.tsx                ← ЗАМЕНИТЬ (дашборд)
│   ├── lib/
│   │   └── data.ts             ← НОВЫЙ (моковые данные + типы)
│   ├── components/
│   │   ├── icons.tsx           ← НОВЫЙ (SVG иконки)
│   │   ├── Sidebar.tsx         ← НОВЫЙ
│   │   ├── ProjectCard.tsx     ← НОВЫЙ
│   │   └── Modal.tsx           ← НОВЫЙ
│   ├── projects/
│   │   └── page.tsx            ← НОВЫЙ (список проектов)
│   ├── project/[id]/
│   │   └── page.tsx            ← НОВЫЙ (страница проекта)
│   ├── visit/[projectId]/[visitId]/
│   │   └── page.tsx            ← НОВЫЙ (страница визита)
│   ├── notifications/
│   │   └── page.tsx            ← НОВЫЙ
│   └── settings/
│       └── page.tsx            ← НОВЫЙ
```

## Быстрый старт

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Страницы

- `/` — Дашборд (статистика, активность, проекты)
- `/projects` — Список проектов с фильтрами
- `/project/1` — Страница проекта с таймлайном визитов
- `/visit/1/1` — Визит: фотосетка, статусы, загрузка фото
- `/notifications` — Уведомления
- `/settings` — Настройки профиля
