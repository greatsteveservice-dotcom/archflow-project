# Archflow — Design System Reference

Документ для Claude Design. Кратко: всё, что нужно, чтобы новые экраны (лендинг, кабинет заказчика и т.п.) совпадали со стилем продукта.

---

## 1. Бренд в одной фразе

**Editorial monochrome.** Газетно-журнальный сериф (small-caps), четыре цвета, никакого радиуса, никаких теней, инверсия на hover. Никаких градиентов, никаких иконок-заглушек, никаких декоративных элементов.

Образ: выставочный каталог архитектурной студии × швейцарская типографика × Bauhaus.

---

## 2. Цвета — РОВНО 4

| Токен | HEX | Назначение |
|-------|-----|-----------|
| `--af-black` | `#111111` | Основной текст, инвертированные блоки, бордеры активных состояний |
| `--af-offwhite` | `#F6F6F4` | Фон карточек, hero-блоков, страницы |
| `--af-border` | `#EBEBEB` | Линии, разделители, неактивные элементы, плейсхолдеры |
| `--af-white` | `#FFFFFF` | Фон страницы (в светлых местах), фон topbar, инверсия на hover |

Дополнительно для текста (НЕ для фонов и бордеров):
- `--ink-muted` rgb(100, 100, 100) — вторичный текст
- `--ink-faint` rgb(150, 150, 150) — третичный
- `--ink-ghost` rgb(200, 200, 200) — placeholder-like

**Запрещено:** красный (`#E24B4A`, `#DC2626`), синий (`#2563EB`), зелёный, любые промежуточные серые (`#999`, `#666`), любые brand-акценты. Эмфаз делается **инверсией**, а не цветом.

Тёмная тема **выключена принудительно** в `theme.tsx` — лендинг и весь продукт всегда светлые.

## 3. Типографика

**Один шрифт на всё:** **Vollkorn SC** (small-caps serif, Google Fonts).

Веса:
- **400** — body, лейблы, кнопки, подписи
- **600** — активное состояние, акцент
- **700** — заголовки секций, карточек, табов
- **900** — заголовок страницы, крупные метрики

⚠️ Веса 300 в Vollkorn SC **нет** — использовать 400.

Vollkorn SC уже рендерит lowercase как small-caps, поэтому `text-transform: uppercase` применяется **только** к мелкой микрокопи ≤ 11px (лейблы, breadcrumbs, индексы блоков, статусы, кнопки 9–11px). На заголовках — никогда.

Letter-spacing для Vollkorn — компактный:
- 9–11px микрокопи: `0.18em`–`0.2em`
- 12–13px кнопки/мета: `0.06em`–`0.08em`
- Заголовки (16px+): `-0.01em` или `0`

### Типографическая шкала
| Уровень | Размер (desktop / mobile) | Вес | Применение |
|---------|--------------------------|-----|-----------|
| L1 page title | 52 / 32 | 900 | Главный заголовок страницы |
| L2 section name | 32 / 22 | 700 | Имя hero-блока |
| L3 section hero | 32 / 28 | 900 | Заголовок раздела внутри проекта |
| L3 tab name | 20 / 19 | 700 | Имя таба-блока |
| Project card name | 24 / 22 | 700 | Имя проекта |
| Project thumbnail | 72 / 56 | 900 | Декоративная цифра в углу карточки |
| Body | 13 | 400 | Основной текст |
| Label / micro | 9 | 400, uppercase, letter-spacing 0.2em | Метки, индексы |

CSS-переменные адаптивных шрифтов: `--af-fs-7` … `--af-fs-13`. На мобиле `input/textarea/select` принудительно `16px !important` (анти-iOS-zoom).

## 4. Базовые правила

- **`border-radius: 0` всегда и везде** — модалки, тултипы, бейджи, аватары, чек-боксы, юнред-точки. Глобальный reset:
  ```css
  *, *::before, *::after { border-radius: 0 !important; box-shadow: none !important; }
  ```
- **`box-shadow: none`** — нигде. Глубина не нужна.
- **Gap между блоками** = `2px` (переменная `--af-gap`). Это даёт фирменный плотный «сетчатый» вид.
- **Бордеры** — всегда `0.5px` или `1px solid #EBEBEB`.
- **Hover = инверсия цвета.** Светлый блок (`#F6F6F4`) → `#111` фон + белый текст. Без анимаций кроме `background/color 0.2s ease`.
- **Кнопки — ghost.** Прозрачный фон + бордер `#111` → на hover заливается чёрным. Никаких filled-primary вроде `bg-blue-600`.
- **Никаких иконок-эмодзи в UI.** Иконки — SVG stroke 2.4, размер 28px, класс `className`, **никогда** inline `style`.

## 5. Компонентные паттерны

### Topbar
Sticky, белый фон, 16×24 padding, нижний бордер `0.5px solid #EBEBEB`. Логотип слева — Vollkorn SC 15px / 700. Контекст справа — micro-label 9px uppercase.

### Hero-блок (большой кликабельный раздел)
```
flex, align-items: flex-end
background: #F6F6F4
padding: 32px 32px
min-height: 160px
hover → background: #111, текст белый
```
Внутри: индекс (9px micro `01`), название (32px / 700), подзаголовок (9px micro), стрелка `→` справа (24px).

### Project card (сетка проектов)
```
2-колонки, gap: 2px
min-height: 180px
background: #F6F6F4
hover → инверсия
```
В правом-верхнем углу — декоративная огромная цифра `01`/`02` (72px / 900, цвет `#EBEBEB`).

### Tab-блок (sub-секции)
2 колонки, gap 2px, `min-height: 120px`, тот же hover pattern. Имя 20px / 700. Активный таб — постоянно в инвертированном состоянии (`#111` фон, белый текст).

### Settings-блок
Тёмный (`#111`) уменьшенный блок (`min-height: 64px`), используется для пунктов «Настройки», «Выйти» и т. п. На hover становится `#F6F6F4`.

### Bottom TabBar
Фикс-бар внизу авторизованных экранов. Белый фон, верхний бордер `2px solid #111`. 4 SVG-иконки (stroke 2.4, 28px). Тот же шрифт.

### Модалки / bottom-sheet
Без рамок, без радиуса, бордер `1px solid #111` или белый фон над затемнением. Bottom-sheet — `height: 88dvh`.

### Формы / инпуты
Прозрачный фон, нижний бордер `1px solid #EBEBEB`, на focus → `#111`. Padding `12px 0`. Лейбл сверху — 9px micro uppercase.

### Кнопки (полный паттерн)
```css
padding: 11px 20px;
background: transparent;
color: #111;
border: 1px solid #111;
font-family: Vollkorn SC; font-size: 9–13px;
letter-spacing: 0.14em; text-transform: uppercase;
hover → background: #111; color: #FFF;
```
Никаких filled / pill / rounded.

## 6. Layout

- Контентный контейнер: `max-width: 960px; margin: 0 auto`.
- Mobile breakpoint: `767px`. Tablet: `768–1023px`. Desktop: `1024+`.
- На мобиле уменьшаются паддинги (24→16) и размеры заголовков (52→32, 32→22).
- Структура страниц всегда вертикальная, plate-on-plate, с `2px` зазорами. Никаких карусельных слайдеров и иконочных рядов.

---

## 7. Образ, отличающий Archflow от типичных SaaS-лендингов

- **Никаких 3D-иллюстраций, абстрактных blob-фигур, цветных градиентов.**
- **Никаких пастельных карточек «фичей» с круглой иконкой сверху.**
- Лендинг должен ощущаться как **печатная обложка журнала об архитектуре**: крупные засечки, монохром, плотные блоки, индексы `01/02/03`.
- Кейсы / скриншоты подаются внутри `#F6F6F4` плашек с тонкой `1px` рамкой, без теней.
- Призыв к действию — ghost-кнопка, не заливная.

---

## 8. Голос (tone of voice)

- Русский, на «вы» в сторону клиента, нейтрально-профессионально.
- Короткие декларативные фразы. Без маркетингового пафоса («революционная платформа»).
- Микрокопи (9px uppercase) — терминологическая: `01 ПРОЕКТЫ`, `STATUS / ACTIVE`, `02 NEW`.
- Заголовки — конкретные: «Авторский надзор без бумаги», а не «Реши все свои проблемы».

---

## 9. Что НЕ менять без явного запроса

1. Шрифт (только Vollkorn SC).
2. Палитру (только 4 цвета).
3. `border-radius: 0`.
4. Структуру навигации.
5. Запрет dark mode.

---

## 10. Дано-структура (для Claude Design)

Когда генерируешь экран:

1. Использовать только цвета из таблицы §2.
2. Шрифт — Vollkorn SC, веса 400/600/700/900.
3. Каждый блок — прямоугольник без скруглений и теней. Между блоками `2px`.
4. Hover-эффект — инверсия цвета (фон ↔ текст).
5. Заголовки — крупные засечки, **lowercase** (small-caps делает Vollkorn SC сам).
6. Микрокопи (9–11px) — UPPERCASE с letter-spacing 0.18–0.2em.
7. Никаких иконок-emoji, никаких filled-кнопок, никаких градиентов.
8. CTA = ghost-кнопка с инверсией.
9. Секции страницы маркировать индексами `01 / 02 / 03` сверху — это фирменный приём.
10. Контейнер — 960px, на мобиле — full-width с `16–20px` боковыми паддингами.
